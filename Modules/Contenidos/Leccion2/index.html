Lección 2: Sintaxis de Java

En esta lección aprenderás la sintaxis básica de Java, paso a paso, con ejemplos claros y prácticos para que puedas empezar a escribir tus primeros programas.

1. Estructura de un programa Java

Todo programa en Java necesita una clase principal y un método main desde donde comienza la ejecución. 

public class MiPrograma {
    public static void main(String[] args) {
        System.out.println("¡Hola Mundo!");
    }
}

- public class MiPrograma define la clase.
- public static void main(String[] args) es el método principal.
- System.out.println() imprime texto en la consola.

2. Comentarios

Los comentarios no se ejecutan y sirven para explicar tu código.

// Esto es un comentario de una línea
/* Esto es un comentario
   de varias líneas */

3. Variables y Tipos de Datos

Las variables almacenan información. Debes indicar el tipo de dato:

  • int: números enteros
  • double: números con decimales
  • boolean: true o false
  • char: un solo carácter
  • String: texto o cadena de caracteres
int edad = 25;
double altura = 1.75;
boolean esEstudiante = true;
char inicial = 'A';
String nombre = "Ana";

4. Operadores

Java tiene operadores para realizar cálculos y comparaciones:

  • + - * / % → Aritméticos
  • == != > < >= <= → Comparación
  • && || ! → Lógicos

5. Programa Completo

public class InfoPersona {
    public static void main(String[] args) {
        String nombre = "Ana";
        int edad = 25;
        double altura = 1.70;

        System.out.println("Nombre: " + nombre);
        System.out.println("Edad: " + edad);
        System.out.println("Altura: " + altura);
    }
}

6. Ejercicio Práctico

Crea un programa que:

  • Declare variables para tu nombre, edad y ciudad.
  • Imprima estos datos en la consola con mensajes claros.

Luego, prueba cambiando los valores y observa cómo se actualiza la salida.
