Lección 1: Instalación

En esta lección, aprenderás a instalar Java Development Kit (JDK), Visual Studio Code y a configurar todo para escribir y ejecutar tu primer programa en Java.

Paso 1: Instalar Java (JDK)

  1. Ve a el sitio oficial de Oracle y descarga la última versión del JDK para tu sistema operativo (Windows, macOS o Linux).
  2. Ejecuta el instalador y sigue los pasos. En Windows, deja las opciones por defecto y finaliza la instalación.
  3. Para comprobar que Java está instalado, abre la terminal (o PowerShell) y escribe: 
    java -version
    Deberías ver la versión instalada. Si no, reinicia tu PC.

Paso 2: Instalar Visual Studio Code

  1. Ve a la web oficial de Visual Studio Code y descarga el instalador para tu sistema operativo.
  2. Instala VS Code siguiendo los pasos del asistente.

Paso 3: Configurar VS Code para Java

  1. Abre Visual Studio Code.
  2. Ve a la sección de extensiones (icono de cuadrados a la izquierda o Ctrl+Shift+X).
  3. Busca e instala la extensión "Extension Pack for Java" de Microsoft. Esto instalará todo lo necesario para programar en Java (soporte de lenguaje, depuración, autocompletado, etc).
  4. Reinicia VS Code si te lo pide.

Paso 4: Crear y Ejecutar tu Primer Programa

  1. Crea una nueva carpeta para tus proyectos Java y ábrela en VS Code (Archivo > Abrir carpeta...).
  2. Dentro de la carpeta, crea un archivo llamado HolaMundo.java.
  3. Escribe el siguiente código: 
    public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("¡Hola, Mundo!");
    }
}
  4. Guarda el archivo. Haz clic en el botón de "play" (▶️) que aparece arriba del método main o haz clic derecho en el archivo y selecciona "Run Java" para ejecutarlo.
  5. Deberías ver el mensaje ¡Hola, Mundo! en la terminal de VS Code.
